
# Technical Portfolio - Mohamad Aidiel Azzaufi

A high-performance, responsive technical portfolio designed for a Technical Support Supervisor. This project showcases a blend of hardware diagnostic expertise and modern web development practices.

## 🚀 Features
* **Dual-Theme Engine:** Seamless switching between Dark (Deep Navy) and Light (Sand/Slate) modes with local storage persistence.
* **Dynamic Navigation:** Interactive scroll-spy that updates active navigation links as you browse.
* **Hardware Aesthetic:** Themed using `Fira Code` and slate palettes to reflect a systems-engineering background.
* **Interactive Footer:** Custom JavaScript eye-tracking logic in the footer pupils.

## 🛠 Tech Stack
* **Frontend:** HTML5, CSS3 (Custom Variables), JavaScript (ES6+).
* **Animations:** AOS (Animate on Scroll) Library.
* **Typography:** Inter & Fira Code (Google Fonts).

## 📂 File Structure
* `index.html` - Main entry point & structure
* `main.js` - Core logic (Theme, Eyes, Scroll)
* `Azzaufi_CV.pdf` - Downloadable resume
* `/images` - Profile photo and assets
